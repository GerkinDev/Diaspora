Class: Entity

EntityFactory. Entity

The entity is the class you use to manage a single document in all data sources managed by your model.

Note that this class is proxied: you may try to access to undocumented class properties to get entity's data attributes

new Entity( [source])

An entity is a document in the population of all your datas of the same type

Create a new entity.

Parameters:
Name Type Argument Default Description
source Object | DataStoreEntities.DataStoreEntity <optional>
 {}

Hash with properties to copy on the new object.
