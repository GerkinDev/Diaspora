Class: Set

Set

Collections are used to manage multiple entities at the same time. You may try to use this class as an array.

new Set(model, entities)

Create a new set, managing provided entities that must be generated from provided model.

Parameters:
Name Type Argument Description
model Model

Model describing entities managed by this set.
entities Entity | Array.<Entity> <repeatable>

Entities to manage with this set. Arguments are flattened, so you can provide as many nested arrays as you want.
