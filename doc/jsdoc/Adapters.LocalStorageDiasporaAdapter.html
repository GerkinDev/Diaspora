Class: LocalStorageDiasporaAdapter

Adapters. LocalStorageDiasporaAdapter

This class is used to use local storage or session storage as a data store. This adapter should be used only by the browser

new LocalStorageDiasporaAdapter( [config])

Create a new LocalStorage data store

Parameters:
Name Type Argument Description
config Object <optional>

Options hash.

Properties
Name Type Default Description
session Boolean false

If false, data source will use local storage. If true, it will use session storage.
